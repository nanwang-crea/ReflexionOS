## 当前问题

1. **工具调用不稳定** - 依赖 Prompt 让模型手写 JSON
2. **执行不够稳健** - 空响应、解析失败时直接结束
3. **总结可能缺失** - 工具执行后可能没有 final summary
4. **无实时反馈** - 看不到 LLM 思考过程和流式输出

## 启动方式

```bash
# 终端1 - 后端
cd backend && uvicorn app.main:app --reload --host 127.0.0.1 --port 8000

# 终端2 - 前端
cd frontend && npm run dev
```

访问: http://localhost:5173
